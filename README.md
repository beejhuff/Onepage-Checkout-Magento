Onepage-Checkout-Magento
========================

Install via Composer
--------------------

```json
"require": {
    ...
    "oggettoweb/onepageCheckout": "1.*"
    ...
},
"repositories": [
    ...
    {
        "type": "vcs",
        "url": "https://github.com/rut4/Onepage-Checkout-Magento"
    }
],
...
"extra": {
    "magento-root-dir": ".",
}

```
